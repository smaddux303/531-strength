# 5/3/1 Marathon Strength

A mobile-first strength training app built around the 5/3/1 program, designed for marathon runners who lift once or twice a week.

## Features

- One-day and two-day split modes
- Auto-calculated weights from training maxes
- Core/ab work between sets
- Pre-workout check-in (fatigue + RPE)
- Training phase tagging (Base / Build / Peak / Taper / Recovery)
- Inline plate calculator — tap any weight
- 1RM estimator
- Cycle history log
- Custom lift names
- Backup & restore (JSON export/import)
- Works offline (PWA)

## Setup

Hosted via GitHub Pages. Add to home screen on iOS (Safari → Share → Add to Home Screen) or Android (Chrome → menu → Add to Home Screen).

## Updating weights

Tap the TM badge on any lift card, or go to ⚙ Settings.
